Ray Tracing (RT)

This package provides a differentiable ray tracer for radio propagation modeling.

The Primer on Electromagnetics provides useful background knowledge and various definitions that are used throughout the API Documentation.

The best way to get started is by going through some of the Tutorials.

Advanced users may want to have a look at the Developer Guides to get a deeper understanding of the inner workings of Sionna RT and how it can be modified or extended according to one’s needs.

Sionna RT is available as a standalone package whose source code is located in a separate GitHub repository. Additionally, the Sionna RT GUI package can be used to simulate and visualize scenes, paths, and radio maps. A technical report describing the algorithms implemented in Sionna RT is available here.
