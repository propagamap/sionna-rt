Sionna RT GUI
=============

.. image:: https://raw.githubusercontent.com/NVlabs/sionna-rt-gui/refs/heads/main/src/sionna_rt_gui/data/preview.webp
    :align: center
    :width: 100 %


An interactive GUI to simulate and visualize Sionna RT scenes, paths, and radio maps.
All simulation parameters can be manipulated via the GUI or YAML configuration files.
Transmitters and receivers can be placed, moved and animated interactively.

Please see the `GitHub repository <https://github.com/NVlabs/sionna-rt-gui>`_ for more information.
